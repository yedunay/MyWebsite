ft_printf

42 Projesi: C dilinde printf fonksiyonunun yeniden yazımı

20 Nisan 2025 Yunus Emre DUNAY

Proje Amacı

ft_printf projesi, C dilinde standart printf fonksiyonunun temel işlevlerini yeniden oluşturmayı amaçlar. Bu sayede format çözümleme, çoklu argüman kullanımı ve buffer optimizasyonu gibi konular üzerine derinlemesine çalışılmış olur.

Kullanılan Teknolojiler

  • C (ISO C90)
  • Variadic Functions (stdarg.h)
  • Makefile
  • Linux terminal debugging (Valgrind, GDB)

Fonksiyon Prototipi

int ft_printf(const char *format, ...);

Desteklenen Formatlar

  • %c - karakter
  • %s - string
  • %d, %i - signed decimal integer
  • %u - unsigned decimal integer
  • %x, %X - hexadecimal
  • %p - pointer
